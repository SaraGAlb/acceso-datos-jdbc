# acceso-datos-jdbc

Este repositorio contiene dos proyectos de ejemplo relacionados con el
acceso a datos usando JDBC:

- `jdbc-basico`: demostración mínima con un único DAO que ejecuta
  consultas CRUD.
- `arquitectura-capas`: aplicación con capas (repositorio, servicio,
  aplicación) que muestra un diseño más robusto y desacoplado.

Cada proyecto es independiente y dispone de su propio `pom.xml` y
README. Consulta el subdirectorio correspondiente para más detalles.
